# Networking Basics for DevOps

## OSI Model
1. Physical
2. Data Link
3. Network
4. Transport
5. Session
6. Presentation
7. Application

## Important Ports

22 - SSH
80 - HTTP
443 - HTTPS
3306 - MySQL
5432 - PostgreSQL

## TCP vs UDP

TCP - Reliable, connection-oriented
UDP - Fast, connectionless

## What is CIDR?

Example:
192.168.1.0/24

Means:
256 IP addresses
