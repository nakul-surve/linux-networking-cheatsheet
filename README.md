# Linux & Networking Cheatsheet for DevOps

This repository contains essential Linux and Networking concepts required for DevOps and Cloud Engineering.

## Topics Covered

- Linux file system & permissions
- Process management
- Networking fundamentals
- Troubleshooting commands
- Common interview questions

## Why I Built This

To strengthen my fundamentals and prepare for real-world production environments.



## Real-World Scenario Example

Scenario: Website is down in production.

Steps I would take:

1. Check if EC2 instance is running
2. Check Security Group inbound rules
3. Verify service is running (systemctl status)
4. Check logs (/var/log/)
5. Validate Load Balancer health checks
6. Confirm DNS resolution

Goal: Reduce MTTR (Mean Time To Recovery)
